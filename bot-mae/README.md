# ミリオンアーサー エクスタシス オートパイロット (bot-mae)


----
## 使い方
1. Seleniumをruby gemでインストール
`gem install selenium-webdriver`
2. p_connect_run.rbの"YOUR_GREE_ID"と"YOUR_GREE_PASSWORD"を書き換え
3. 起動
`ruby mae_run.rb`
